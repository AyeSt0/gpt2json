# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows semantic versioning while it is distributed as releases.

## [0.1.0] - 2026-04-27

### Added

- Protocol-first OAuth export flow.
- Concurrent batch processing engine.
- Sub2API and CPA JSON export writers.
- Lightweight PySide6 desktop UI.
- Input parser registry with auto-detect entry point.
- Backend-first mailbox OTP provider registry skeleton.
- GitHub-ready project metadata, tests, CI, and documentation.
