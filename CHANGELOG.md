# Changelog

本项目使用类似 [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) 的格式记录重要变化。

## [0.1.0] - 2026-04-27

### Added

- 协议优先 OAuth 导出流程。
- 并发批处理引擎。
- Sub2API / CPA JSON 导出。
- PySide6 轻量桌面 GUI。
- 输入格式 parser 注册表与 auto-detect 入口。
- Backend-first 邮箱 OTP 注册表骨架，围绕 IMAP / Graph / JMAP / POP3 / API 扩展。
- GitHub-ready 项目元数据、测试、文档和模板。
