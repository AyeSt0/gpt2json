# 支持与反馈

## 推荐渠道

- Bug：使用 [Bug 反馈](https://github.com/AyeSt0/gpt2json/issues/new?template=bug_report.yml)。
- 功能建议：使用 [功能建议](https://github.com/AyeSt0/gpt2json/issues/new?template=feature_request.yml)。
- 使用问题：先查看 [`docs/troubleshooting.md`](docs/troubleshooting.md)。

## 反馈时请提供

- GPT2JSON 版本；
- 使用 GUI 还是 CLI；
- 输入格式类型；
- 期望导出格式（Sub2API / CPA / 两者）；
- 脱敏后的日志片段或 `failure_report.safe.json` 中的分类；
- 可复现的合成示例。

请不要提供真实账号、密码、token、cookie、验证码、导出 JSON 或邮箱正文。
